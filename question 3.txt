A. What is reverse-i-search in a Linux terminal?
reverse-i-search: Search backward starting at the current line and moving ‘up’ through the history as necessary. 
Activate reverse-i-search using Ctrl+r and then type in a query to find matches. Hit Ctrl+r again to find the next match.

B. How do you list out the hidden directories using Linux command line tool?
Command: ls -a
The ls command lists the contents of the current directory. The –a switch lists all files – including hidden files.
